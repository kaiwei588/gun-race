# 如何开启在线游玩（解决 404）

你的代码已经在 GitHub 上，只需 **手动开启 Pages** 一次。

## 操作步骤（跟着做）

1. **打开这个链接**（仓库 Pages 设置）  
   👉 https://github.com/kaiwei588/gun-race/settings/pages

2. 找到 **Build and deployment**

3. **Source（来源）** 选择：**Deploy from a branch**

4. **Branch（分支）** 选择：
   - 分支：`main`
   - 文件夹：`/ (root)`

5. 点击 **Save（保存）**

6. 等 **2～5 分钟**，页面顶部会出现绿色提示：  
   `Your site is live at https://kaiwei588.github.io/gun-race/`

7. 再打开：**https://kaiwei588.github.io/gun-race/**

---

## 常见问题

**Q：Settings 里没有 Pages 选项？**  
→ 确认仓库是 **Public（公开）**，私有仓库需 GitHub Pro 才能用 Pages。

**Q：保存后还是 404？**  
→ 多等几分钟，或强制刷新（Ctrl+F5 / Cmd+Shift+R）。

**Q：Source 选成了 GitHub Actions？**  
→ 改回 **Deploy from a branch → main → / (root)** 更简单。
